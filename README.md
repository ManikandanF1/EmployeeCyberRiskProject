# Employee Cyber Risk Monitoring & Awareness Platform

## Project Description
This project simulates a basic SOC (Security Operations Center) monitoring dashboard.  
It detects file changes, modification timestamps, and file sizes to identify suspicious activity inside employee systems.

## Real-World Use Case
- Detect unauthorized file creation (malware, data exfiltration tools)
- Monitor employee system file changes
- Raise awareness about insider threats

## Technologies Used
- Python
- Flask Web Framework
- HTML/CSS
- OS File Monitoring

```bash
pip install flask
